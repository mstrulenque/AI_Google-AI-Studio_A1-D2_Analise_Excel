>#### OBJETIVO:

 - Realizar a Analise dos 10 sites mais acessados e criar uma conclusão sobre como está sendo usado o tempo.
   
>#### UTILIZANDO: 

 - Google AI Studio
 - Model - Gemini 1.5 Pro
 - Arquivo excel contendo o histórico do Chrome

>#### REFERENCIA:

 - AULA 01 - DESAFIO 02
